About

Welcome to the Academic Evaluation System, an interactive web application that helps you analyze and predict student math scores based on a comprehensive dataset. Our platform leverages the power of machine learning to provide valuable insights and predictions for academic performance.

DESCRIPTION

The Academic Evaluation System is designed to assist students, parents, and educators in understanding and predicting math scores by considering various factors that may influence a student's academic achievements.Our platform provides a comprehensive analysis of each model's performance, enabling you to choose the most suitable one for your prediction needs.

KAGGLE DATASET

This web application utilizes a dataset sourced from Kaggle, which includes information about students' demographics, family background, and test preparation.

Model Results and Prediction

We evaluate several machine learning models to predict math scores. Our platform provides a comprehensive analysis of each model's performance.Based on our model evaluations, the Linear Regression model emerges as the most accurate choice for predicting math scores.
